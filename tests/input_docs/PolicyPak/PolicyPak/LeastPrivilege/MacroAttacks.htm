Does Endpoint Privilege Manager block Macro attacks?

Partially, yes, but this is by design.

First, Microsoft already has protections in to properly block Macro attacks themselves. For more information on this, please see the Microsoft article, New feature in Office 2016 can block macros and help prevent infection.
