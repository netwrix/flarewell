Endpoint Policy Manager for Thunderbird

If you've got zillions of desktops, how are you going manage things like: Update behavior (turning it off, so users aren't prompted) Ensure Thunderbird is always the default mail client? Ensure that spell-check is turned on? Ensure Anti-virus protection is turned on? Prevent users from picking up web cookies? Or anything else that's important to your Thunderbird configuration? Answer: Without Netwrix Endpoint Policy Manager (formerly PolicyPak), you've got no enterprise-way to deliver these settings and lock down Thunderbird. Watch this video to get a handle on the problem and where Endpoint Policy Manager can ease the pain.
