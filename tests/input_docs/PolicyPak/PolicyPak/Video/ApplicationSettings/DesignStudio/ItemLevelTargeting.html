Predefined ILTs (Internal Filters)

Using build 545 and later, you can use "Internal Filters" to specify when your Pak should work on specific machines. For instance, you might only want an applications settings to hit the machine *WHEN* the application is actually on the machine. This video is for Pak designers only. For normal "Item Level Targeting" (https://www.policypak.com/pp-blog/item-level-targeting)filters, see our Video on that function (for day to day Endpoint Policy Manager administration.)
