Deploying Endpoint Policy Managerdirectives without Group Policy (Endpoint Policy Manager Exporter Utility)

You might want to avoid using Group Policy to deliver Netwrix Endpoint Policy Manager (formerly PolicyPak) directives if you are using Intune, SCCM, LanDesk, KACE or similar software for software deployment, and your team doesn't want to use Group Policy but wants to use Endpoint Policy Manager. See this video to see how it's done.
