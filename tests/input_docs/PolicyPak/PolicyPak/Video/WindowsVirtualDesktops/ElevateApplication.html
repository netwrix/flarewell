Endpoint Policy Manager + WVD: Elevate application inside WVD and bypass UAC prompts

Sometimes and app won't run unless you're running with local admin rights. NOT a good idea with WVD. Instead, use Netwrix Endpoint Policy Manager (formerly PolicyPak) Least Privilege Manager to elevate applications when needed, and keep Standard Users just that... users and NOT admins.
