SecureCopy(TM). Empower users to copy then elevate items

Netwrix Endpoint Policy Manager (formerly PolicyPak) SecureCopyâ„¢ enables you to specify a specific UNC share (or shares) for users from which to copy .EXEs, .MSIs, and Scripts. Users can then RIGHT-CLICK and use a command to begin the copy of the item and finally place the item in target locations on the local computer.

The net result is that applications which need writable locations to perform their installs or runs may do so in an elevated state; where they would not able to run within the UNC share.
