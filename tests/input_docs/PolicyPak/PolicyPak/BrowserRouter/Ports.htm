Routing Using Ports

Endpoint Policy Manager Browser Router can open a specific website when a particular website needs a specific port. In the example below we have www.portquiz.net:1001 being used for a specific Google Chrome website.

NOTE: For a video on using Endpoint Policy Manager Browser Router and ports, see Endpoint Policy Manager Browser Router and Ports.

In this way, you can have granular control over which browser is opened for which website.
