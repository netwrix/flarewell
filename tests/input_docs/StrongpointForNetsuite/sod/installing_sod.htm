Installing the SoD Library Bundle

  1. Open Customization > SuiteBundler > Search & Install Bundles

  2. Under KEYWORDS, type 311216.

  3. Click Search.

  4. SelectStrongpoint SoD Library


  5. Install the Bundle as usual.

  6. Review the Default SoD Custom Record Types and Permission Lists table and add any permissions needed to your Custom Roles.
