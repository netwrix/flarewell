Get Pending Requests

Use this API to retrieve information about all those requests with pending status provided you have the required permissions to manage all requests. See the Miscellaneous section of the Security Role – Permissions topic.

Endpoint

https://machinename:4443/GroupIDDataService/api/IdentityStores/{identityStoreId}/Workflows/Pending

HTTP Methods

GET.
