Get Metadata of Data Source by Server Type and ID

Use this API to retrieve metadata information of all data sources defined so far in Directory Manager by File Server type (i.e., local server or a cloud server) and by data source type (i.e., MS Excel or Text/CSV or MS Access or SQL server).

Endpoint

https://machinename:4443/GroupIDDataService/api/DataSource/GetMetaDataByType/{serverType}/{datasourceType}

HTTP Method

GET
